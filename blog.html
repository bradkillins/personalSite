Practice, Practice, Practice!
Nov 19, 2019

Watching tutorials and following examples in class are great ways to learn new concepts, but if you want that newly minted information to still be in your head three days (or three hours) later....PRACTICE!

It might seem obvious, but it bears repeating here. For the longest time I was following online video tutorials but nothing ever seemed to stick.

Taking what you learn in a video and practicing that concept in your own way is what made concepts start to stick for me. If you want to up your retention even more, the best way (other than more practice) it to help others learn. When explaining a concept to someone else, you'll realise your own short-comings very quickly.

CSS Media Queries are AWESOME
Nov 18, 2019

The first time I used media queries in CSS my mind was blown! If you're unfamiliar, media queries allow you to conditionally apply CSS stylings. The most widely used condition to check is the width of the user's screen.
If you want a responsive site, media queries are essential.

@media screen and (max-width: 800px) { }
With this one simple line you can have styling (code goes in {} ) that will only apply when the user's screen is 800px or less.
This is a very common way of using media queries, however there are many other conditions to check for.
Visit MDN - CSS @media for a full description, including formal syntax and multiple examples.
