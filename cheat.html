CSS Cheat-sheet

Selectors

* { } - all elements
div { } - all div tags
div, p { } - all divs and paragraphs
div p { } - paragraphs inside divs
div > p { } - all p tags, one level deep in div
div + p { } - p tags immediately after div
div ~ p { } - p tags preceded by div
.classname { } - all elements with class
#idname { } - element with ID
div.classname { } - divs with certain classname
div#idname { } - div with certain ID
#idname * { } - all elements inside #idname

Linking

External stylesheet:
<link rel="stylesheet" href="styles.css" />
Inline:
Don't use inline styling!
Internal:
Use <style> tags in head.

Flex Box

display: flex; - Use flex layout.
flex-direction: row; - Displays items horizontally.
flex-direction: column; - Displays items vertically.
flex-wrap: wrap; - Items can wrap to next row.
flex-wrap: nowrap; - Items stays on a single row.
justify-content: center - Align items in center of row.
justify-content: space-around; - Puts space around each item in row.
align-content: center; - Align items in center of column.
align-content: space-between; - Only space between items.

Transform

translateX(); - moves item along x-axis.
translateY(); - moves item along y-axis.
scaleX(); - scales item along x-axis.
scaleY(); - scales item along y-axis.
rotate(); - rotates item - use deg as unit

Media Queries

Conditionally apply styling:
@media screen and (max-width: 480px) { }
This will apply styling only if viewport is 480px or less.
